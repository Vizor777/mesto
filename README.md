# Практическая работа №4

## Описание проекта
"Mesto" - интерактивной страницей, куда можно добавлять фотографии, удалять их и ставить лайки.

## Используемые технологии:
* Nested БЭМ;
* Резиновая верстка;
* Адаптивная верстка;
* Использование media queries;
* Работа с ветками (branches) в гите;
* Flex Layout;
* Grid Layout;
* JS;
* HTML Form.
